# 📚 AI-Assisted Book Management System

A production-ready backend application built using Java, Spring Boot, and MySQL that enables secure book catalog management, issue/return workflows, and AI-powered library assistance.

---

## 🚀 Features

- User Authentication (JWT)
- Role-Based Authorization
- Book Catalog Management
- Author Management
- Category Management
- Book Issue & Return
- RESTful APIs
- Swagger Documentation
- Docker Support
- AI Book Recommendation
- AI Book Summary
- Natural Language Book Search

---
